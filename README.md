
# Star Resume

- An easy to use Web application for creating great resume for interviews. 

- Great for creating a single page resume for interviews.

- Deployed on - https://taupe-alpaca-3d6e99.netlify.app/



## Tech Stack 

- React Js
- HTML
- JavaScript
- CSS
